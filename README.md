# Núcleo de Esportes (Backend)

## Tecnologias

- [GoLang](https://go.dev/doc/tutorial/getting-started)
- [Supabase](https://supabase.com/dashboard/projects)

## Instalação 

Clone o repositório:

```sh
git clone https://github.com/nucleo-de-esportes/backend.git
cd backend
```

## Executando o projeto

Compilar código:
```sh
go build ./...
```

Executar:
```sh
go run ./...
```
